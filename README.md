This is the repository for the frontend code.

# Setting up the project
- Make a copy of .env.example named .env and then copy your API key into it. To obtain the API key containing our data, ask Raymond. Alternatively, you can set the backend API url to localhost:3000 if you want to use a local copy of the backend.
- Run the command `npm install` in the project directory.

Running the project locally
- run the command `npm run dev` in the project directory. Then go to the localhost url provided.
